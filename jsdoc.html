CodeSpells Documentation

JavaScript Functions

These JavaScript functions are essentially part of the "CodeSpells API". The following code will list the functions in an organized fashion, with all their known parameters.

Control:

function onCreate()
function onHit()
function onActivate()
orb.setTimeout("function", milliseconds)
orb.setInterval("function", milliseconds)
orb.createOrb('spell name', Earth Mana, Water Mana, Air Mana, Fire Mana)
orb.destroySelf()

Movement:

orb.orbitPlayer(Distance, Time Per Revolution)
orb.orbitOwner(Same parameters as above)
orb.disableOrbit()
orb.lockToPlayerRotation(Distance)
orb.unlockFromPlayerRotation()
orb.turnLeft(Degrees)
orb.setRotationUp()
orb.setRotationDown()
orb.setRotationRandom()
orb.pointAtPlayer()

Properties

getLifeTime()
distanceToPlayer()
getEarthMana()
getWaterMana()
getFireMana()
getAirMana()
setColor('Color Hex Value')

World Objects:

getPlayer()
getCreator()
getLastHitWorldObject()
isMoveable()
getMass()

Earth Powers:

liftSand(true/false)
lowerSand(true/false)
createPhysicalConnection(Connected Object, "STRENGTH(IN CAPS)")
breakPhysicalConnections()
moveWorldObjectToOrb(Object)
changeRock(value)

Water Powers:

setPushWater('True/False', Strength Value)
setAlterWaterFlow('True/False', Strength Value, Direction offset in degrees)
setRemoveWater('True/False')
setHumidity('True/False', Humidity Value)
setAddWater('True/False*')

Fire Powers:

fire_explosion()
setOrbTemperature(Value)

Debug

log('MESSAGE/WARNING/ERROR', 'Message')
evaluateUnityScript('UnityScript')

Note: Most of these functions work with orb. behind it. For example, orb.fire_explosion()

Note: If you are trying to find one object but it has to be separated with a ".", adding parentheses will make it one object. For example, orb.getLastHitWorldObject().getMass() would have to be changed to (orb.getLastHitWorldObject()).getMass() to find the mass of the last hit object.
